# BMI calc
## Installing
### 1. Clone the repo
  ``` bash
  git clone https://github.com/keyunge/bmi-cli.git
  ```
### 2. Install a compiler
  #### Debian/Ubuntu

  ``` bash
  sudo apt install clang
  ```

  #### Fedora

  ``` bash
  sudo dnf install clang
  ```

  #### Arch

  ``` bash
  sudo pacman -Sy clang
  ```
### 3. Run install script
  ``` bash
  cd bmi-cli
  chmod +x install.sh
  ./install.sh
  ```
### 4. Run the program
  ``` bash
  bmi-cli
  ```
## Usage
  1. Type ```bmi-cli``` in your terminal
  2. Enter your height and weight
  3. Check your bmi
  4. See the funny message :)
